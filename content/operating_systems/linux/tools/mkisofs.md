[mkisofs](https://linux.die.net/man/8/mkisofs) is a utility that creates an ISO 9660 image from files on disk.
<!--more-->

## Usage

```shell
mkisofs -o DVD_DISC.iso -dvd-video -V MY_DISC2 "/directory/containing/VIDEO_TS/folder/"
```
