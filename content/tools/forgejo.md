[Forgejo](https://forgejo.org/)  is a self-hosted, lightweight software forge (git repository manager) that enables 
developers to manage code with built-in features like issues, pull requests, and CI/CD. 
<!--more-->
It is a community-driven fork of Gitea, focused on transparent, non-profit development.
Forgejo was created in late 2022 because a for-profit company, Gitea Ltd, took control of the Gitea project without community consensus.

## Installation...

### ...in Kubernetes

Forgejo can be installed into a Kuberentes cluster using the [Forgejo Helm Chart](https://code.forgejo.org/forgejo-helm/forgejo-helm)
