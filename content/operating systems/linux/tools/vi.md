`vi` is a screen-oriented text editor originally created for the Unix operating system.
<!--more-->

## Handy Commands

| Command | Description                     |
|---------|---------------------------------|
| Esc     | Switch to Command mode          |
| `dd`    | Delete the line                 |
| `i`     | Switch to Insert mode           |
| `G`     | Go to the last line in a file   |
| `:q!`   | Quit vi and do not save changes |
| `:wq`   | Save and Quit                   |
